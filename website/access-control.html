Access Control

Authentication

Cantaloupe has built-in support for HTTP Basic authentication. To enable it, set the following keys in the configuration file:

http.auth.basic = true
http.auth.basic.username = myusername
http.auth.basic.secret = mypassword

When enabled, the entire website will be restricted.

HTTP Basic transmits passwords in clear text, so it should only be used on a trusted local network, or in conjunction with SSL/TLS.

A reverse proxy server can be used to support other means of authentication.

Authorization

Currently, Cantaloupe is not authorization-aware.

About

Cantaloupe is an open-source image server developed at the University of Illinois at Urbana-Champaign Library. By combining the basic operations of cropping, scaling, rotation, flipping, color filtering, and format conversion, it can support on-the-fly derivative image generation and deep zooming into high-resolution source images.

The IIIF Image API is an open standard intended to promote interoperability between clients — like zooming image viewers — and image servers. Cantaloupe supports versions 1.1 and 2.0 of this API. Most of its processors meet the highest compliance level, and should work with any IIIF-compatible client software.

Cantaloupe features a configurable processing pipeline based on:

  • Resolvers, that locate and retrieve source images;
  • Processors, that transform them according to the parameters in an image request; and
  • Caches, that cache generated images.

This makes it adaptable, while remaining easy to configure and use.

Features

  • Easy to get working
  • Self-contained
  • Pluggable resolvers for filesystem, database, HTTP, and Amazon S3 sources
  • Pluggable processors to support a variety of source image formats, including JPEG, TIFF, and JPEG2000 — as well as still-frame extraction from videos in a variety of formats
  • Configurable server- and client-side caching
  • Multiple endpoints
  • Available built-in authentication

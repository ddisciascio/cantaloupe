Endpoints

IIIF Image API 1.1

Compliance

Cantaloupe dynamically computes and declares its compliance level with this API on a per-processor basis. Most processors are "Level 2"-compliant.

Location

This endpoint is available at /iiif/1.1.

Enabling/Disabling

This endpoint can be enabled or disabled by setting endpoint.iiif.1.1.enabled to true or false in the configuration file.
